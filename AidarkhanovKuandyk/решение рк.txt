Задание №2
Образует ли мультипликативную группу ненулевые матрицы специального вида, состоящие из вещественных чисел
Для того, чтобы доказать образует ли множество группу нужно доказать ряд свойств:
1. замкнутость
2. ассоциативность 
3. наличие нейтрального элемента
4. наличие обратного элемента
Доказательство:
1. при умножении двух матриц с вещественными числами будет матрица с вещественными числами
2. (а*б)*с=а*(б*с)
Возьмем за а,b,c матрицы: 
[(x  y) (a  b)] (c  d) = ( ax-yb   bx+ya)(c  d) = acx-byc-dbx-ady  adx-dby+cbx+cya
[(-y x) (-b a)] (-d c)   (-ay-bx  -by+ax)(-d c)	 -acy-bcx-dax+dby -ady-dbx+acx-cby

( x y) [( a b)(c d) ] = (ac-bd ad+bc ) (x y) = acx-bdx-ady-bcy acy-bdy+adx+bcx
(-y x) [(-b a)(-d c)]	(-bc-ad -bd+ac)(-y x) -bcx-adx-acy+bdy -bcy-ady+acx-bdx  - ассоциативность выполняется

3. существует такой e -> х*е=х, е*х=х
Возьмем аналог "1" в матричной форме - единичную матрицу
(x  y)(1 0) = (х  y) 
(-y x)(0 1)   (-y x)
Единичная матрица является нейтральным элементом группы G

4. для любого х всегда существует y, такой что x*y=e, y*x=e
В качестве x возьмем матрицу:
(х у)
(-у х)

х*х^-1=1(E)

x^-1=x^T/detX
detX=x^2+y^2 
x^T=(x -y) -> x^-1 = (x -y)*1/(x^2+y^2)
    (y x)            (y x)
